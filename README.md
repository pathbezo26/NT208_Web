## 📌 Project Overview
A **WhatsApp-like Chat System** built for a Web / Software Engineering course project.

The goal is to simulate a modern real-time messaging platform with core chat features and a scalable architecture.

Current planned features:
- 🔹 Private chat
- 🔹 Group chat
- 🔹 Message history
- 🔹 Message ordering
- 🔹 Delivery guarantee
- 🔹 Real-time communication with Socket.IO / WebSocket
- 🔹 Concurrent users testing
- 🔹 Message queue integration (planned)

---

## 🛠️ Tech Stack
- **Frontend:** React + CSS
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **Realtime:** Socket.IO
- **Protocol:** WebSocket

> ✅ You can absolutely mention **WebSocket** in README.
> It is not a programming language, but it is an important **core communication technology** of the project.
